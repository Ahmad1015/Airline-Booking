# Airline Booking System

This is a simple Java program for an Airline Booking System. It allows passengers to book seats in either First Class or Economy Class, depending on seat availability.

- [Usage](#usage)
- [Features](#features)

## Introduction

The Airline Booking System is designed to manage seat reservations for a flight. It provides options for booking seats in First Class or Economy Class, and it ensures that bookings are within the available seat limits. If all seats in both classes are booked, it notifies the user that the next flight is in 3 hours.

## Usage

To use this program, follow these steps:

1. Clone the repository to your local machine:

   ```shell
   git clone https://github.com/Ahmad1015/Airline-Booking.git
2. Compile the Java program:
  ```shell
  javac AirlineBooking.java
```
3. Run the program:
   ```shell
   java AirlineBooking
4.Follow the on-screen instructions to book seats in First Class or Economy Class.

If all seats are booked, you will be informed that the next flight is in 3 hours.
## Features
- Booking seats in First Class or Economy Class.
- Managing seat availability.
- Providing boarding passes with seat numbers.
- Handling cases when a class is full and offering alternatives.
